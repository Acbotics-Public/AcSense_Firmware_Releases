To access released versions of the AcSense Firmware, go to the releases page.

https://github.com/Acbotics-Public/AcSense_Firmware_Releases/releases

For instructions on flashing firmware onto the AcSense see [Customer AcSense Programming Procedure](Customer%20AcSense%20Programming%20Procedure.pdf)

For connecting the PicKit to the AcSense you will need a 100726 adapter cable. These can be ordered from Acbotics or made based on the drawing here. [100726 Picoblade 8 to SL 8.pdf](100726%20Picoblade%208%20to%20SL%208.pdf)
